## 🚀 파이썬 제어문 학습 정리 (Control Flow Statements)

파이썬의 \*\*제어문(Control Flow Statements)\*\*은 코드의 실행 흐름을 원하는 대로 조작하는 데 사용됩니다. 크게 **조건문**과 **반복문**으로 나뉩니다.

---

### 1\. 조건문 (Conditional Statement): `if`

**`if` 문**은 특정 **조건**이 \*\*참(True)\*\*일 때만 코드 블록을 실행하도록 합니다.

#### 📝 기본 형태

```python
if 조건식1:
    # 조건식1이 참일 때 실행되는 코드 블록
    실행문A
elif 조건식2:
    # 조건식1이 거짓이고, 조건식2가 참일 때 실행되는 코드 블록
    실행문B
else:
    # 위의 모든 조건식이 거짓일 때 실행되는 코드 블록
    실행문C
```

#### ✨ 핵심 특징

- **들여쓰기(Indentation)**: 파이썬은 중괄호 `{}` 대신 \*\*들여쓰기(공백 4칸 또는 탭)\*\*를 사용하여 코드 블록을 구분합니다. 조건문 뒤에는 반드시 콜론(`:`)을 사용해야 합니다.
- **`elif` (Else if)**: 여러 개의 조건을 차례로 검사할 때 사용합니다. `elif`는 필요한 만큼 여러 번 사용할 수 있습니다.
- **`else`**: 모든 조건이 거짓일 때 실행되며, 생략 가능합니다.
- **조건식**: 결과가 **불리언(Boolean)** 값(`True` 또는 `False`)인 표현식이나 함수가 들어갑니다. (예: `A > 10`, `name == 'Alice'`)

#### 💡 예제

```python
score = 85

if score >= 90:
    print("A 학점")
elif score >= 80:
    # 90 미만이고 80 이상일 때 실행
    print("B 학점")
else:
    print("C 학점 이하")
```

### 2\. 반복문 (Loop Statement): `while`

**`while` 문**은 특정 **조건**이 \*\*참(True)\*\*인 동안 코드 블록을 **반복해서** 실행합니다.

#### 📝 기본 형태

```python
while 조건식:
    # 조건식이 참인 동안 반복적으로 실행되는 코드 블록
    실행문
```

#### ✨ 핵심 특징

- **조건**: `while` 루프가 실행될 때마다 조건식을 검사합니다. 조건식이 **거짓**이 되는 순간 반복이 종료됩니다.
- **무한 루프 방지**: 루프 내부에는 반드시 **조건식을 변경**시켜 반복을 언젠가 종료시킬 수 있는 코드가 포함되어야 합니다. 그렇지 않으면 \*\*무한 루프(Infinite Loop)\*\*에 빠지게 됩니다.

#### 💡 예제

```python
count = 0
while count < 3:
    print(f"현재 횟수: {count}")
    # 조건을 변경하는 코드 (필수)
    count += 1

# 출력:
# 현재 횟수: 0
# 현재 횟수: 1
# 현재 횟수: 2
```

### 3\. 반복문 (Loop Statement): `for`

**`for` 문**은 **반복 가능한(Iterable)** 객체 (리스트, 튜플, 문자열, 딕셔너리, 집합 등)의 **모든 요소**를 하나씩 순회하며 코드 블록을 실행합니다.

#### 📝 기본 형태

```python
for 변수 in 반복_가능한_객체:
    # 반복_가능한_객체의 각 요소가 '변수'에 할당되며 실행
    실행문
```

#### ✨ 핵심 특징

- **`range()` 함수**: **정해진 횟수**만큼 반복할 때 가장 많이 사용되는 함수입니다.
  - `range(n)`: 0부터 $n-1$까지의 정수를 생성합니다.
  - `range(start, stop)`: `start`부터 `stop-1`까지의 정수를 생성합니다.
  - `range(start, stop, step)`: `start`부터 `stop-1`까지 `step` 간격으로 정수를 생성합니다.
- **딕셔너리 순회**: 딕셔너리를 `for` 문으로 순회하면 기본적으로 \*\*키(Key)\*\*가 할당됩니다. 키와 값 모두를 얻으려면 `.items()` 메서드를 사용합니다.

#### 💡 예제

**1. 리스트 순회**

```python
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(f"{fruit}는 맛있다.")
```

**2. `range()`를 이용한 반복 횟수 지정**

```python
for i in range(5):  # 0, 1, 2, 3, 4
    print(i)
```

**3. 딕셔너리 순회**

```python
person = {'name': 'Alice', 'age': 30}
for key, value in person.items():
    print(f"{key}: {value}")
```

### 🧩 모든 제어문에 사용되는 보조 키워드

|     키워드     | 역할                                                                                | 사용 예시                                         |
| :------------: | :---------------------------------------------------------------------------------- | :------------------------------------------------ |
|  **`break`**   | 반복문(for, while)을 **즉시 종료**하고 루프 다음 코드로 넘어갑니다.                 | `if condition: break`                             |
| **`continue`** | 반복문의 **나머지 코드 실행을 건너뛰고**, 다음 반복(cycle)으로 즉시 넘어갑니다.     | `if condition: continue`                          |
|   **`pass`**   | 문법적으로 코드가 필요하지만 **아무것도 하지 않도록** 처리합니다.                   | `if condition: pass` (나중에 코드를 채울 때 유용) |
|   **`else`**   | **`for` 또는 `while`** 루프가 **`break` 없이 정상적으로 완료**되었을 때 실행됩니다. | `for i in range(5): ... else: print("완료")`      |
