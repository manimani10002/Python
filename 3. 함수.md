### 1\. 함수(Function)의 기본 이해

#### 함수란 무엇인가? 🧱

함수란 **특정한 작업을 수행하도록 설계된 코드 블록**입니다. 입력값을 받아 정해진 규칙에 따라 처리하고, 그 결과를 반환할 수도 있습니다.

#### 함수를 사용하는 이유는 무엇일까? 🎯

1.  **재사용성(Reusability)**: 한 번 정의한 코드를 여러 곳에서 반복적으로 호출하여 사용할 수 있습니다.
2.  **모듈화(Modularity)**: 복잡한 프로그램을 기능별로 나누어 관리하고 개발할 수 있어 가독성이 높아집니다.
3.  **유지보수(Maintenance)**: 기능 수정이 필요할 때 해당 함수 블록만 고치면 되므로 유지보수가 편리해집니다.

---

### 2\. 파이썬 함수의 구조 및 사용법

#### 파이썬 함수의 구조 🛠️

함수는 `def` 예약어를 사용하여 정의하며, 콜론(`:`)과 들여쓰기로 실행 블록을 구분합니다.

```python
def 함수이름(매개변수1, 매개변수2):
    # 실행할 코드
    return 반환값
```

#### 매개변수(Parameter)와 인수(Argument) 🗣️

- **매개변수 (Parameter)**: 함수를 정의할 때 함수 내부에서 사용하기 위해 설정하는 **변수** (위 구조에서 `매개변수1, 매개변수2`).
- **인수 (Argument)**: 함수를 호출할 때 실제로 함수에 전달하는 **값**입니다.

#### 입력값과 반환값에 따른 함수의 형태 🔄

|               형태               | 정의           |      `return` 키워드      | 예시                                |
| :------------------------------: | :------------- | :-----------------------: | :---------------------------------- |
|        **일반적인 함수**         | 입력 O, 반환 O |           사용            | `def add(a, b): return a + b`       |
|      **입력값이 없는 함수**      | 입력 X, 반환 O |           사용            | `def get_pi(): return 3.14`         |
|      **반환값이 없는 함수**      | 입력 O, 반환 X | 생략 또는 `return`만 사용 | `def print_name(name): print(name)` |
| **입력값도, 반환값도 없는 함수** | 입력 X, 반환 X |           생략            | `def hello(): print("Hello")`       |

### 3\. 함수의 입력값 처리 고급 기법

#### 매개변수에 초깃값 미리 설정하기 ⚙️

함수 정의 시 매개변수에 미리 기본값을 할당해두면, 함수 호출 시 해당 인수를 생략해도 기본값이 사용됩니다. (단, 기본값이 설정된 매개변수는 기본값이 없는 매개변수 뒤에 위치해야 함)

```python
def greet(name, msg="안녕하세요"):
    return f"{name}님, {msg}"
```

#### 매개변수를 지정하여 호출하기 (키워드 인수) 🔑

함수를 호출할 때 **매개변수 이름**을 지정하여 인수를 전달하는 방식입니다. 인수의 순서에 상관없이 값을 전달할 수 있습니다.

```python
def sub(a, b): return a - b
result = sub(b=5, a=10) # 5
```

#### 입력값이 몇 개가 될지 모를 때 (가변 인수) 🎁

인수의 개수가 정해지지 않았을 때 사용합니다.

- **일반 위치 기반 인수 (`*args`)**: 함수 호출 시 전달되는 모든 위치 기반 인수를 **튜플** 형태로 받습니다.

  ```python
  def sum_all(*args):
      return sum(args)
  ```

- **키워드 매개변수 (`**kwargs`)**: 함수 호출 시 전달되는 모든 키워드 인수 (`key=value` 형태)를 **딕셔너리** 형태로 받습니다.

  ```python
  def print_info(**kwargs):
      for key, value in kwargs.items(): print(f"{key}: {value}")
  ```

### 4\. 함수 내 변수의 효력 범위와 관리

#### 함수 안에서 선언한 변수의 효력 범위 🌐

- 함수 안에서 선언된 변수는 \*\*지역 변수(Local Variable)\*\*로, 함수 호출이 끝나면 메모리에서 사라집니다. 함수 밖에서는 접근할 수 없습니다.

#### 함수 안에서 함수 밖의 변수를 변경하는 방법 💡

함수 밖의 \*\*전역 변수(Global Variable)\*\*를 함수 내에서 변경하려면 특별한 조치가 필요합니다.

1.  **`return` 사용하기**: 변경된 값을 반환받아 외부 변수에 다시 할당하는 가장 일반적이고 권장되는 방법입니다.
2.  **`global` 명령어 사용하기**: 함수 내에서 `global 변수명`을 선언하면, 이 변수가 함수 외부의 전역 변수임을 명시하고 직접 값을 변경할 수 있습니다. (잦은 사용은 코드의 복잡성을 높일 수 있어 주의 필요)

### 5\. 기타 고급 기능

#### 함수의 반환값은 언제나 하나이다 ☝️

파이썬 함수는 문법적으로 **하나의 값**만 반환합니다. 만약 여러 값을 쉼표로 구분하여 반환하면, 그 값들은 **하나의 튜플**로 묶여서 반환됩니다.

```python
def get_coords():
    return 10, 20 # (10, 20) 튜플 하나가 반환됨
```

#### `lambda` 예약어 (람다 함수) 🚀

`lambda`는 **간단한 함수**를 **익명**으로, 그리고 **한 줄**로 정의할 때 사용하는 예약어입니다. 복잡한 로직이나 여러 줄의 코드가 필요할 때는 일반 함수를 사용해야 합니다.

```python
# 일반 함수: def add(a, b): return a + b
# 람다 함수:
add = lambda a, b: a + b
print(add(3, 5)) # 8
```

#### 함수의 독스트링(Docstring) 📖

함수 정의 바로 아래에 큰따옴표 3개(`"""..."""`) 안에 작성하는 설명 문구입니다. 함수가 어떤 기능을 하고, 매개변수와 반환값이 무엇인지 등에 대한 정보를 담습니다. 이는 개발자에게 유용한 정보를 제공하며, `help(함수이름)` 명령어나 IDE에서 함수 사용법을 확인할 때 표시됩니다.

```python
def calculate(x):
    """
    입력된 x에 2를 곱하여 반환하는 함수입니다.
    :param x: 숫자 (int 또는 float)
    :return: x * 2의 결과
    """
    return x * 2
```
