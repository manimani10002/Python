### 1\. 사용자 입력 활용하기 (Input)

#### `input()` 사용하기 💻

파이썬의 내장 함수인 \*\*`input()`\*\*은 사용자로부터 데이터를 입력받는 데 사용됩니다. `input()` 함수가 반환하는 값은 종류에 관계없이 \*\*항상 문자열(string)\*\*입니다.

```python
# 사용자에게 입력 요청 후, 결과는 'result' 변수에 문자열로 저장됨
result = input()
```

#### 프롬프트를 띄워 사용자 입력받기 💬

`input()` 함수 안에 문자열을 인수로 넣으면, 그 문자열이 사용자에게 표시되는 **프롬프트** 역할을 합니다.

```python
# 프롬프트 메시지가 사용자에게 표시됨
name = input("이름을 입력하세요: ")
print(f"안녕하세요, {name}님.")
```

#### 입력값을 숫자로 변환하기 🔢

`input()`의 결과는 항상 문자열이므로, 산술 연산을 수행하려면 반드시 숫자로 \*\*형 변환(Type Conversion)\*\*을 해야 합니다.

- **정수로 변환하기**: `int()` 함수 사용
  ```python
  age_str = input("나이: ")  # 입력값: "25"
  age_int = int(age_str)     # 결과: 25 (정수)
  ```
- **실수로 변환하기**: `float()` 함수 사용
  ```python
  pi_str = input("원주율: ") # 입력값: "3.14"
  pi_float = float(pi_str)   # 결과: 3.14 (실수)
  ```
- **한 번에 변환하기**: `input()` 함수를 형 변환 함수 안에 넣어 중첩해서 사용하면 코드가 간결해집니다.
  ```python
  num1 = int(input("첫 번째 숫자: "))
  ```

### 2\. `print()` 자세히 알기 (Output)

#### 큰따옴표로 둘러싸인 문자열은 `+` 연산과 동일하다 ➕

`print()` 함수 내에서 문자열을 **쉼표(`,`)로 구분**하면, 각 문자열 사이에 **자동으로 공백**이 추가되어 출력됩니다. 이 방식은 문자열을 `+` 연산자로 연결하는 것과는 내부적으로 다릅니다 (`+` 연산은 실제로 새로운 문자열을 생성).

```python
print("Hello", "World") # 출력: Hello World
print("Hello" + " " + "World") # 출력: Hello World
```

#### 문자열 띄어쓰기는 쉼표로 한다 ➡️

`print()` 함수에서 여러 인수를 쉼표로 구분하여 전달하는 것이 기본적으로 공백을 넣어 출력하는 가장 간단한 방법입니다.

#### `sep` 매개변수로 구분자 설정하기 🚧

`print()` 함수의 **`sep` (separator)** 매개변수를 사용하면, 출력되는 인수들 사이에 들어갈 **구분자**를 직접 지정할 수 있습니다. 기본값은 공백(`' '`)입니다.

```python
print("2025", "12", "25", sep="-")
# 출력: 2025-12-25
```

#### 한 줄에 결괏값 출력하기 (end 매개변수) 🎯

`print()` 함수는 기본적으로 출력을 마치면 \*\*줄바꿈(`\n`)\*\*을 합니다. **`end`** 매개변수를 사용하면 줄바꿈 대신 원하는 문자열을 출력의 끝에 넣을 수 있습니다. 기본값은 `'\n'`입니다.

```python
print("A", end="")
print("B")
# 출력: AB (A와 B가 한 줄에 붙어서 나옴)

for i in range(5):
    print(i, end=" ")
# 출력: 0 1 2 3 4
```

### 3\. 실습: 간단한 계산기 만들기 🧮

오늘 배운 `input()`, `int()`, `print()`, 그리고 `sep`을 활용하여 간단한 덧셈 계산기를 구현합니다.

```python
# 1. 사용자로부터 두 숫자를 정수로 입력받기
num1 = int(input("첫 번째 정수를 입력하세요: "))
num2 = int(input("두 번째 정수를 입력하세요: "))

# 2. 덧셈 계산
result = num1 + num2

# 3. 결과를 sep과 함께 출력
print("결과:", num1, "+", num2, "=", result, sep=" ")
# 예시 출력: 결과: 10 + 20 = 30
```
